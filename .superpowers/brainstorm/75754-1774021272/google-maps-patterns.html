Key Changes from Google Maps Pattern

✓ Adopted
  • Single panel — detail replaces list, not side-by-side
  • Circular action buttons in horizontal row
  • Compact info rows with icons
  • Collapse arrow to maximize map
  • Back navigation to return to list
  • Comments collapsed — count shown, click to expand
✗ Not Adopted (BTC Map specific)
  • No photo — BTC Map doesn't have merchant photos
  • No ratings/stars — not a review platform
  • No tabs — not enough content to warrant tab navigation
  • Keep Bitcoin-specific — payment methods, boost, verify as first-class
