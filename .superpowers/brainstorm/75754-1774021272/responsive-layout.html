Breakpoint Logic

≥ 1536px (2xl): List (320px) + Drawer (380px) + Map (rest) → both panels visible
768px – 1535px (md–xl): Single panel (350px) + Map → detail replaces list
< 768px (mobile): Full map + bottom sheet drawer (no change to current)
